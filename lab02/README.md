# Лабораторная работа №2  
## Моделирование изменения температуры в пластине на основе одномерного уравнения теплопроводности

Программа разработана на языке Python с использованием графического интерфейса Tkinter.  
В ходе работы программы была заполнена таблица значений температуры в центральной точке пластины после 2 секунд модельного времени.

### Таблица 1 – Значения температуры в центральной точке пластины (°C)

| Шаг по времени, с \ Шаг по пространству, м | 0.1 | 0.01 | 0.001 | 0.0001 |
|-------------------------------------------|-----|------|-------|--------|
| **0.1** | 50,0000 | 25,0454 | 24,1501 | 24,1406 |
| **0.01** | 50,0000 | 24,7772 | 23,8209 | 23,8107 |
| **0.001** | 50,0000 | 24,7488 | 23,7860 | 23,7757 |
| **0.0001** | 50,0000 | 24,7460 | 23,7824 | 23,7722 |

### Выводы

*Было страшно ждать расчёт для комбинации 0.0001 × 0.0001!*

На основе представленных данных о температуре в центральной точке пластины после 2 секунд модельного времени можно сделать следующие выводы:

1. **Проблема грубой дискретизации**: при шаге по пространству 0.1 м во всех случаях получается значение 50°C, что соответствует температуре правой границы. Это указывает на недостаточную точность расчёта из-за слишком грубой пространственной дискретизации (всего два узла сетки на всей длине пластины).

2. **Сходимость по пространству**: при уменьшении шага по пространству наблюдается устойчивая сходимость решения — температура монотонно убывает и стремится к определённому предельному значению. Например, при фиксированном шаге по времени 0.0001 с уменьшение шага по пространству с 0.01 до 0.0001 м приводит к изменению температуры с 24,746°C до 23,7722°C.

3. **Сходимость по времени**: аналогичная картина наблюдается и при уменьшении шага по времени. При фиксированном dx = 0.001 м температура изменяется с 24,1501°C (dt=0.1) до 23,7757°C (dt=0.0001). Разница между результатами при dt=0.001 и dt=0.0001 составляет менее 0.01°C, что свидетельствует о хорошей сходимости по времени.

4. **Оптимальные параметры расчёта**: наиболее точное решение **23,7722°C** получено при наименьших шагах dx=0.0001 м и dt=0.0001 с. Для практических расчётов с точностью до 0.01°C достаточно использовать шаги **dx=0.001 м** и **dt=0.001 с**, что даёт значение температуры **23,786°C**.

5. **Итог**: численный метод демонстрирует устойчивую сходимость как по пространству, так и по времени, что подтверждает корректность реализации алгоритма решения одномерного уравнения теплопроводности методом конечных разностей.
