# Отчет по лабораторной работе №5

## Тема: Программная реализация стохастических моделей с графическим интерфейсом пользователя (GUI)

### 1. Цель работы
Изучение методов интеграции базового датчика случайных чисел в прикладное программное обеспечение. Реализация двух моделей случайных событий: бинарного выбора («Да/Нет») и многовариантного предсказания («Magic 8-Ball») с использованием библиотеки Tkinter.

### 2. Описание алгоритма и математической модели
В данной работе использован ранее разработанный **Линейный конгруэнтный генератор (LCG)**. Для выбора случайного события из списка реализован следующий алгоритм:

1.  **Генерация:** Вызов метода `next()`, возвращающего число $\alpha$ в диапазоне $[0, 1)$.
2.  **Масштабирование:** Определение индекса $k$ в массиве предсказаний длиной $m$.
    $$k = \lfloor \alpha \cdot m \rfloor$$
    Использование функции `int()` в Python эквивалентно математической операции взятия целой части ($\lfloor \dots \rfloor$).
3.  **Логический фильтр:** Введение приоритетного условия для обработки специфических текстовых запросов (детерминированный выход).

### 3. Программная реализация

#### 3.1. Структура классов
* `SimpleLCG`: Класс генератора. Параметры $a=1103515245$, $c=12345$, $m=2^{31}$, что обеспечивает высокое качество псевдослучайной последовательности.
* `MysticBallApp`: Основной класс приложения, управляющий жизненным циклом GUI и логикой анимации.

#### 3.2. Ключевые функции
* **Метод `shake_ball`**: Реализует имитацию физической тряски объекта. Использует рекурсивные вызовы `after()` для изменения координат `ball_canvas`, что предотвращает "зависание" интерфейса во время вычислений.
* **Метод `ball_logic`**: 
    - Извлекает строку из `Entry`.
    - Проверяет наличие триггер-фразы *"радмир ренатович"*.
    - При обнаружении триггера устанавливает $answer = \text{"ДА"}$.
    - В противном случае выбирает ответ на основе $k$-го индекса из текущего набора (`preds_yes_no` или `preds_magic`).
* **Метод `animate_text`**: Создает эффект плавного появления текста за счет итеративного изменения цвета шрифта в массиве `["#1a1a2e", "#00f2fe", "#ffffff"]`.


### 6. Выводы
В ходе выполнения работы была подтверждена эффективность использования ЛЦГ для создания пользовательских приложений. Реализованная модель "Магического шара" наглядно демонстрирует переход от непрерывного равномерного распределения $[0, 1)$ к дискретным событиям. Разработанный графический интерфейс с элементами анимации и динамического изменения состояний объектов подтверждает корректность работы событийного цикла (MainLoop) в среде Python.
