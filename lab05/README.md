
# Отчет по лабораторной работе №5
## Тема: Разработка стохастических приложений с графическим интерфейсом на базе LCG

### 1. Цель работы
Создание интерактивного приложения для моделирования случайных событий с использованием собственного базового датчика случайных чисел. Реализация двух режимов: бинарного выбора (Да/Нет) и классического «Магического шара».

### 2. Реализация базового датчика случайных чисел
В программе использован линейный конгруэнтный генератор (класс `SimpleLCG`) со следующими параметрами:
* **Модуль ($m$):** $2^{32}$
* **Множитель ($a$):** $1\,664\,525$
* **Приращение ($c$):** $1$
* **Зерно ($Seed$):** текущее системное время (`time.time()`).

Алгоритм вычисляет значения по формуле $X_{n+1} = (aX_n + c) \pmod m$. Для получения числа в интервале $[0, 1)$ результат делится на $m$.

### 3. Алгоритм моделирования событий
Для преобразования непрерывного числа $\alpha \in [0, 1)$ в дискретный индекс события $k$ применяется формула:
$$k = \text{int}(\alpha / \pi), \text{ где } \pi = 1.0 / m$$
Здесь $m$ — количество вариантов в списке. Если результат вычислений превышает границы массива (что возможно при $\alpha \approx 1$), индекс принудительно устанавливается на последнее значение ($m-1$).


### 4. Выводы
В ходе лабораторной работы была успешно реализована интеграция математической модели ЛЦГ в пользовательское приложение. Выбранные параметры генератора ($a=1664525$, $m=2^{32}$) обеспечивают достаточную степень случайности для бытовых предсказательных моделей. Применение методов `after()` позволило реализовать неблокирующую анимацию, что обеспечило высокую отзывчивость интерфейса при обработке стохастических процессов.
