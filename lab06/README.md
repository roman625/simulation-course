## Часть 1. Моделирование дискретной случайной величины (ДСВ)

### 1.1. Алгоритм реализации
Для генерации ДСВ в методе `dsv_generator`  реализован **алгоритм на основе ступенчатой функции распределения**:
1. Вычисляется массив кумулятивных вероятностей $C_k = \sum p_i$ .
2. Генерируется базовое случайное число $\alpha \in [0, 1]$ .
3. Производится поиск интервала, в который попало число $\alpha$: $C_{k-1} < \alpha \le C_k$ .

### 1.2. Статистический анализ
Проверка гипотезы о соответствии распределения проводилась по критерию Пирсона. В коде  реализована формула:
$$\chi^2_{эмп} = \sum_{i=1}^{k} \frac{(n_i - N p_i)^2}{N p_i}$$
где $n_i$ — фактическое количество выпадений значения, $N p_i$ — теоретически ожидаемое.

## Часть 2. Моделирование нормальной случайной величины (НСВ)

### 2.1. Алгоритм реализации
Для генерации НСВ $N(\mu, \sigma)$ в методе `box_muller_norm` использовано **преобразование Бокса-Мюллера**. Оно позволяет трансформировать два независимых равномерных числа $u_1, u_2$ в стандартную нормальную величину:
$$z_0 = \sqrt{-2 \ln(u_1)} \cdot \cos(2\pi u_2)$$
Итоговое значение: $X = \mu + z_0 \cdot \sigma$.

### 2.2. Группировка и критерий согласия
Для проверки непрерывной величины использовалась группировка данных по интервалам.
1. Число интервалов $k$ выбрано по **формуле Стерджеса**: $k = 1 + \log_2 N$ .
2. Теоретическая вероятность попадания в интервал рассчитывалась через функцию распределения `norm.cdf` .
3. Сравнение $\chi^2_{эмп}$ с критическим значением $\chi^2_{0.95}$ .

## Выводы по работе

1. **Точность моделирования:** Реализованные алгоритмы (интервальный метод для ДСВ и Бокса-Мюллера для НСВ) демонстрируют высокую точность. С увеличением объема выборки $N$ погрешность оценки параметров стремится к нулю.
2. **Подтверждение гипотез:** Во всех проведенных опытах значение статистики $\chi^2_{эмп}$ оказалось меньше критического порога $\chi^2_{крит}$. Это математически доказывает, что генерируемые последовательности соответствуют заданным законам распределения на уровне значимости **0.05**.
